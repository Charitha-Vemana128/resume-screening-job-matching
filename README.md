## Resume Screening & Job Matching System

### Description
This project focuses on the conceptual design of an NLP-based system to compare resumes with job descriptions.

### Technologies
Python, NLP, TF-IDF, Cosine Similarity, Scikit-learn

### Approach
- Text preprocessing of resumes and job descriptions
- Feature extraction using TF-IDF
- Similarity measurement using cosine similarity
- Candidate ranking based on similarity score

### Scope
The project emphasizes understanding the NLP workflow and similarity logic rather than full implementation.
